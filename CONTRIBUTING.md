# Contributing to COTA

Thank you for your interest in helping build Camps On The Air (COTA).

COTA is currently an open community project. The goal is to create a fun, accessible amateur radio activity that encourages people to combine radio, camping, and outdoor exploration.

At this stage, the most valuable contributions are ideas, discussion, and feedback.

## How you can help

You do not need to be a programmer to contribute.

We welcome help from:

- Amateur radio operators
- Campers and outdoor enthusiasts
- Portable operation enthusiasts
- Web developers
- Database contributors
- Designers
- Documentation writers
- People with ideas and feedback

## Ways to contribute

### Share ideas

Open a discussion or issue with ideas about:

- Activation rules
- Award categories
- Eligible locations
- VHF/UHF operation
- Backcountry activation
- Logging systems
- Website features

### Improve documentation

Help make COTA easier to understand by improving:

- Rules
- FAQs
- Guides
- Explanations

### Technical contributions

If you have experience with:

- Web development
- Databases
- Mapping systems
- Applications
- Automation

your help would be greatly appreciated.

## Before contributing

Please remember:

COTA is still in the concept stage. Nothing is finalized.

The goal is to build something that is:

- Accessible
- Fun
- Community-driven
- Welcoming to new and experienced operators

All constructive feedback is welcome.

Thank you for helping shape COTA.
